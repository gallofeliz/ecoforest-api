# ecoforest-api

See docker-compose.yml ; option ECOFOREST_USERNAME/PASSWORD if proxy (without auth) is used

Call /summary and have

```
{
    temperature: 19.1,
    power: 5,
    status: 7,
    humanStatus: 'running',
    mode: 'power',
    targetPower: 5
}
```

Others endpoints are available to control the stove.
